# Calculator-Practice-Project

A practice project for juniors to learn GitHub collaboration by improving a basic calculator.

## Project Goal

This project will help students learn:

* GitHub workflow
* Collaboration on real projects
* Improving existing code

## Tasks for Contributors

Level 1

* Improve calculator UI

Level 2

* Add percentage (%) button

Level 3

* Add dark mode

Level 4

* Add keyboard input support

Level 5

* Add calculation history

## How to Contribute

1. Fork this repository
2. Clone the forked repository
3. Create a new branch
4. Add your feature or improvement
5. Create a Pull Request
